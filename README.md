# Front-end Development Challenge

This is a technical challenge to test Front-end development-related skills.
## Installation

Simply npm install.

```bash
npm i
```

## Usage

### Development server

```bash
npm run dev
```

You can view the development server at `localhost:8080`.

### Testing

```bash
npm run test:unit
```
### Production build

```bash
npm run build
```
